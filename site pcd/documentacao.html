Clique Tab para navegar entre páginas

Nós criamos uma ferramenta com jquery que ao clicar “tab” abre um menu para uma pessoa navegar entre o conteúdo da página, para isso nós criamos primeiro o menu no html e personalizamos da maneira que queríamos no css, depois disso no js criamos uma variável chamada xTriggered que tem o valor de 0, então xTriggered = 0, depois colocamos para o menu ficar com o ‘hide’ para ele começar sem aparecer na tela do usuário, então $(“#menu”).hide, depois colocamos uma função para quando a pessoa clicar uma tecla algum evento acontecer, $(‘body’).keydown(function (event) { } ), depois dentro dessa função criamos uma condição para quando a pessoa clicar tab a variável xTriggered ganhar mais 1 de valor, $(‘body’).keydown(function (event) { if ( event.which == 9 ) { xTriggered++ } } ), e por fim criamos uma verificação para quando a pessoa clicar 1 ou 2 vezes o tab o menu fica aberto e poder alternar entre as 2 opções que tem no menu, $(‘body’).keydown(function (event) { if ( event.which == 9 ) { xTriggered++ } If ( xTriggered < 3 && xTriggered > 0){ $(#menu).show() } Else {} } ) No else colocamos para quando o “xTriggered” for apertado 2 vezes o menu irá desaparecer, então o código ficou assim: Var = xTriggered = 0; $(“#menu”).hide; $(‘body’).keydown(function (event) { if ( event.which == 9 ) { xTriggered++ } If ( xTriggered < 3 && xTriggered > 0 ){ $(#menu).show() } Else { $(#menu).hide() } } )
